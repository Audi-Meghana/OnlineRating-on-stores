# OnlineRating-on-stores
StorePulse is a full-stack MERN platform that allows customers to discover local stores, leave ratings &amp; reviews, and help improve trust among small businesses. Admins manage stores &amp; users, store owners track their ratings, and normal users can rate shops easily.
